# My Poster: AI 포스터 제작 앱

**My Poster**는 **Google 나노 바나나 API**를 활용해 당신을 위한 포스터를 자동으로 만들어주는 앱입니다.

사진 한 장만 있으면, AI가 당신을 분석하여 영화, 콘서트, 이벤트 등 다양한 스타일의 포스터 주인공으로 만들어 줍니다. 복잡한 디자인 지식 없이도 누구나 쉽고 재미있게 **나만의 포스터**를 만들 수 있습니다.
